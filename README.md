🇺🇸 英文描述 (English Description)
AI-Powered Amazon Listing Agent is a next-generation automation tool designed for cross-border e-commerce sellers. Moving beyond traditional keyword stuffing, this software fully embraces the latest trend of AEO (Answer Engine Optimization), perfectly aligning with Amazon's core algorithms: A9 (Search Indexing), COSMO (Intent Knowledge Graph), and Rufus (Generative AI Assistant).

This tool pioneers a Multi-Agent Workflow architecture, breaking down complex operational tasks into three specialized AI nodes to maximize efficiency and conversion rates:

🌟 Core Features
🤖 Multi-Agent Collaborative Pipeline

Data Cleansing (Powered by glm-5): Automatically strips HTML tags, gibberish, and irrelevant ad copy from raw supplier data (e.g., Alibaba/1688), precisely extracting hard physical parameters and facts.

Native Localization (Powered by kimi-k2.5): Translates the structured Chinese facts into natural, native-sounding English tailored for Western e-commerce contexts.

Listing Generation (Powered by qwen3.5-plus): Synthesizes facts, competitor pain points, and SEO keywords to draft highly-converting, AEO-optimized, and strictly compliant Amazon Listings.

🛡️ Real-Time Compliance Guardrail

Built-in blacklist for high-risk promotional or medical claims (e.g., "Cure", "Best Seller").

Features real-time frontend monitoring—any prohibited word instantly triggers a red-border highlight alert, safeguarding your seller account from suspension.

🎯 Surgical Single-Point Rewriting

Exclusive targeted rewrite functionality. Dissatisfied with a specific Bullet Point? No need to regenerate the whole page. With a single click, the AI independently refines and compliance-checks that specific section.

⚡ Seamless & Rapid UX

Supports one-click parsing of local TXT/CSV files for rapid data input.

Offers real-time frontend rendering with options to export the entire listing as a JSON file or one-click copy the raw text directly to Amazon Seller Central.

Target Audience: Amazon Brand Sellers, E-commerce Operations Teams striving for ultimate productivity.
